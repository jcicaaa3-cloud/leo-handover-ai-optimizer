# Disclaimer

This repository is a public portfolio prototype.

The dataset is generated from a simplified synthetic LEO satellite-network simulation. It does **not** contain real satellite operator logs, customer data, proprietary telecom traces, gateway telemetry, commercial deployment data, or production validation results.

The model and API are intended to demonstrate problem framing, simulation design, feature engineering, model evaluation, and deployment-style packaging. They are **not** intended to control or optimize a real satellite or telecom network.

A real industrial system would require high-fidelity orbital propagation, measured RF/channel behavior, gateway and backhaul constraints, latency budgets, fail-safe handover policies, monitoring, security review, and validation with operational data.
